年轻的时候，一般有两种爱情
2019-10-08 3 min read # 情感

我看到过这样一段话：年轻的时候，一般有两种爱情，一种是那种高烧100°C的高温爱情，爱他爱到不行，发高烧。还有一种是长期低烧的爱情，就是你老是在想他到底爱不爱你，你感觉不到他的爱意，就这么赤眉红眼地耗着，耗到你筋疲力尽。

能给女孩子幸福的爱情，不是高烧也不是低烧，而是这种36.8℃的爱情，恒温，平和。幸福是什么？不需要你去燃烧自己，也不需要你低烧，真正的幸福不需要炫耀，幸福，就是偷着乐。

其实，保持恒温说简单也不简单，说难也不难，用最真实的自己和对方交往，并把这种真实和自然变成两个人相处的习惯就好。

恋爱脑的本质是强烈的身份焦虑
2017-02-12 2 min read # 情感

我觉得恋爱脑的本质是强烈的身份焦虑。

英国作家阿兰德波顿在研究人类的身份焦虑时，认为人类天性存在着自我确认的本能。

所谓的自我确认，是人对自身从身体外貌到精神世界，从三观品性到为人处世等全方位的确认需求——我们需要与外界的联结来确认自己的个体存在，确认自己不是无关紧要的“他者”。

而在所有的人际关系中，爱情关系是最直接也最强烈的途径，可以让两个人充分的互相裸露，互相观照，从而使个体可以充分且全面的自我确认，自我认知，自我定位。

这种充分的确认会带给人强烈的愉悦感——我们在爱情里可以充分认可自己及对方的存在，充分肯定彼此的独特性和重要性。

没有给过别人的噢，一直都给你留着
2017-02-01 1 min read # 情感

我是一个在感情里占有欲很强，嘴巴很硬，又很没有安全感的人。

因为太喜欢你了，所以总是幼稚地用各种笨拙的方法来试探你喜不喜欢我，以至于有时候我都觉得自己烦。因为太喜欢你了，所以不乐意任何异性加你的社交软件，或者给你发闲聊的信息。我一边吃醋一遍生闷气，又一边装作若无其事的样子和你说：我很好，我OK。

我知道这样子爱人其实很幼稚，但我在喜欢的人面前，真的成熟不起来，永远是个敏感又自私的贪心小鬼。

我不仅想要被爱，还想要被你偏爱。

就好像并不是最甜的糖才最好吃，最好吃的永远都是你唯一抛出来的那一颗。

我认识这么一个男孩
2016-02-01 4 min read # 情感

我认识这么一个男孩。

他曾对我说过自己对于跟前任的分开特别后悔不已，说自己当时真的不沉稳。

觉得恋爱谈到后期女孩儿总是事事都过分细致，是很难招架的捆绑。“我有段时间做了小手术，不能吃辣，她怕我难受，餐餐陪我吃清粥小菜，我说没必要的，她也不听。当时我觉得窒息。”

他提分手后的前几个月，真的松了一口气。又可以不被人管着，“啊你赶紧起床吃饭”这种。

但在那之后，他参加了保研的夏令营。激烈的竞争焦虑让人快速成长，他在那个夏天经常失眠，反思自己大学前几年的学习生涯，顺便也想起很多恋爱的小事。

这种时期总是很能让人成长，而人成长过后再回过头去看以前的亲密关系，总是会有截然不同的领悟。

男孩也终于知道，就算女孩的关心在当时的当下让他无法接受，他也理应有更好的解决方式，不是逃。
